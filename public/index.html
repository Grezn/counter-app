個人值班工作台（On-call）

你不是一個人在戰鬥：先看告警、再看儀表板、最後照 runbook 處理。
1) 告警中心（PagerDuty） 2) 系統儀表板（Grafana） 3) Logs / APM（Datadog） 4) 基礎設施（AWS） 5) 版本與部署（GitHub） 6) Runbook / 交班筆記
值班小 checklist：
  • 先確認影響範圍（單一服務 / 全站）。
  • 先止血（rollback / failover），再找 root cause。
  • 每 15–30 分鐘更新一次狀態與處置紀錄。
